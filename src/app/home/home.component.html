What is this?

This is a Angular project I created and connected to my E-commerce database. This project is basically a copy of my actual E-commerce site which is built in ReactJS. This project is intended to help me expand my skills with Angular and Typescript.

Why use Angular?

Angular is a more streamlined framework that allows programmers to focus on simply building JavaScript classes. Views and controllers are replaced with components, which can be described as a refined version of directives.
